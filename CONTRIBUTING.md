# OpenMemory

Thank you for your interest in contributing to OpenMemory!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/openmemory.git`
3. Create a virtual environment: `python -m venv venv`
4. Activate it: `source venv/bin/activate`
5. Install dev dependencies: `pip install -e ".[dev]"`

## Development Workflow

1. Create a branch: `git checkout -b feature/my-feature`
2. Make your changes
3. Run tests: `pytest tests/`
4. Format code: `black openmemory/`
5. Commit: `git commit -m "feat: Add my feature"`
6. Push: `git push origin feature/my-feature`
7. Create a Pull Request

## Code Style

- Follow PEP 8
- Use type hints
- Add docstrings
- Write tests

## Reporting Issues

- Use GitHub Issues
- Describe the bug/feature
- Include code examples
- Add system info
